Boba Glitched 1: Data Bending Exploration

Project Overview

Boba Glitched 1 is the first piece in an experimental digital art series focusing on glitch art through manual file manipulation, commonly referred to as data bending.

Tools & Technique

  • Tools: TextEdit (Mac), Preview
  • Method: Open JPEG in TextEdit → copy/paste ~20% of file data → open in Preview to view glitch effects.
  • File Type: JPEG image

Concept & Exploration

This piece experiments with digital image corruption to explore the aesthetics of glitches, creating unexpected visual distortions while preserving enough of the original image to remain recognizable.
