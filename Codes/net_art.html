Net Art Github: 10-Page Digital Portfolio

Project Overview

This project represents a 10-page interactive portfolio designed and hosted on GitHub Pages. Each page showcases different artwork, presentations, and creative explorations from my college career, functioning as a complete online archive of my design and media work.

Category:

Web Development / Digital Art

Tools Used:

GitHub, HTML, CSS, VS Code

Scope:

10 Linked Pages

Design Process & Intent

I designed and coded each page directly in HTML and CSS using Visual Studio Code, then published the site on GitHub Pages for public viewing. The site structure emphasizes clean navigation, consistent styling, and visual storytelling.

Each section of the portfolio highlights a different type of creative work, including digital art, motion design, and experimental projects. The process helped me refine my skills in both web design and presentation layout.

The experience also served as a reflection on my artistic growth — transforming my past academic projects into a cohesive and professional online archive.
